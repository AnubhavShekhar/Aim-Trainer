# Aim Trainer

## Description
Aim Trainer is a simple Python game built using the Pygame library. The game aims to improve your aiming skills by challenging you to click on randomly appearing targets within a limited time frame.

## Installation
To run the game, you need to have Python and Pygame installed on your system. Follow these steps to install the required dependencies:

1. **Install Python**: If you don't have Python installed, download and install it from the [official Python website](https://www.python.org/).
2. **Install Pygame**: Once Python is installed, open your terminal or command prompt and run the following command to install Pygame:

  ```bash
   pip install pygame
  ```

## Usage
After installing the required dependencies, you can run the game by executing the `aim_trainer.py` file. Follow these steps to play the game:

1. Navigate to the directory where the `aim_trainer.py` file is located.
2. Run the following command in your terminal or command prompt:

  ```bash
   python aim_trainer.py
  ```
3. The game window will open, and you can start playing immediately.
4. Click on the targets as they appear on the screen to score points.
5. Keep track of your time, hits, and accuracy displayed on the game window.

## Controls
- Use the mouse to aim and click on targets.
- Press `ESC` or close the game window to exit the game.

## Additional Information
- The game includes features such as displaying game statistics, including time, hits, speed, and accuracy.
- Targets will appear randomly on the screen, and you need to click on them before they disappear.
- The game ends when you miss a certain number of targets, indicated by the remaining lives.
